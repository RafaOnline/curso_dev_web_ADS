# Entrada de dados: variáveis de requisição HTTP

### Características
- Aplicações dinâmicas, utilização de formulários;
- Bankline, rede sociais, plataformas de e-commerce, etc;
- Os usuários acessam as páginas, inserem login e senha e, depois, veem os seus dados e seus conteúdos

### Métodos Get e Post

Método **Get**:
- Os dados são anexados no endereço;

Método **Post**:
- Os dados são enviados no corpo da mensagem;
- o endereço não é alterado;

  <img width="541" height="300" alt="image" src="https://github.com/user-attachments/assets/22ad6841-1703-46a1-820f-caafd705b551" />

  Observação: é importante ficar atento quanto ao método de envio.

### Campos de formulário
- Cada campo de formuláio tem um nome;
- É importante distinguir o nome do campo do seu conteúdo;
- A aplicação do servidor extrai os conteúdos vindos da requisição;
- Os nomes dos campos devem ser os mesmos, tanto do lado cliente como no lado servidor;
