# Exceções em PHP

**Características**

*   É um transtorno quando sistemas param de forma intempestiva;
*   Podem causar muitos danos, sejam financeiros ou até risco à vida;
*   É importante evitar que problemas como esse ocorram;
*   Deve-se utilizar tratamento de exceções;
*   Exceções são comportamentos atípicos dos sistemas;
*   PHP tem recursos para implementar tratamento de exceções.

**Conclusão**

*   O tratamento de exceções em PHP são uma poderosa ferramenta para gerenciar erros e eventos inesperados durante a execução de um programa;
*   O mecanismo de exceções oferece uma forma estruturada e controlada de lidar com erros, separando a lógica de tratamento de erros do fluxo principal do código;
*   Quando ocorre uma condição de erro, uma exceção é "lançada", interrompendo o fluxo normal do programa e transferindo o controle para um bloco de código designado para "capturar" e tratar a exceção;
*   Isso permite que os desenvolvedores lidem com diferentes tipos de erros de maneira específica e organizada.
