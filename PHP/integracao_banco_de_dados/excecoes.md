# Exceções em PHP

**Características**

*   É um transtorno quando sistemas param de forma intempestiva;
*   Podem causar muitos danos, sejam financeiros ou até risco à vida;
*   É importante evitar que problemas como esse ocorram;
*   Deve-se utilizar tratamento de exceções;
*   Exceções são comportamentos atípicos dos sistemas;
*   PHP tem recursos para implementar tratamento de exceções.
