# A Classe PDO

**Características**

*   Há várias linguagens de programação, assim como sistemas de gerenciamento de bancos de dados;
*   Podemos trabalhar em projetos que lidem com diferentes abordagens;
*   É importante conhecer como integrar uma aplicação a um banco de dados;
*   Realização das operações CRUD (**Create, Read, Update e Delete**);
*   A classe PDO tem recursos para facilitar o trabalho de conexão de aplicações desenvolvidas com PHP a bancos de dados.

**Conclusão**

*   A classe PDO ( **PHP Data Objects** ) é uma extensão do PHP que fornece uma interface consistente para acessar diversos sistemas de gerenciamento de banco de dados (SGBD), como MySQL, PostgreSQL, SQLite, entre outros;
*   É uma abordagem orientada a objetos que visa simplificar e unificar a interação com diferentes bancos de dados, independentemente do tipo de SGBD utilizado;
*   Em resumo, a classe PDO é uma ferramenta poderosa e essencial para desenvolvedores PHP que buscam criar aplicações seguras, eficientes e fáceis de manter, oferecendo uma interface padronizada e independente do banco de dados em uso.

## Os métodos da Classe PDO

**Características**

*   A classe PDO tem recursos para facilitar a conexão com bancos de dados;
*   Há métodos prontos da classe PDO para realizar tais tarefas;
*   A conexão é fundamental para que se possa realizar as operações CRUD (*Create, Read, Update, Delete*);
*   Não basta somente estabelecer a conexão com o banco de dados;
*   É necessário que as aplicações possam interagir com o banco de dados;

**Conclusão**

*   PDO (*PHP Data Objects*) fornece uma interface consistente para acesso a diferentes bancos de dados;
*   Simplifica a mudança entre diferentes sistemas de gerenciamento de banco de dados (MySQL, PostgreSQL, SQLite, etc.) sem alterar o código de acesso ao banco;
*   Oferece recursos que melhoram a segurança das aplicações;
*   Há recursos para implementar tratamento de exceções;
*   Suporte a conexões persistentes, reduzindo a sobrecarga de conexões repetidas ao banco de dados.
