    <body>

        <li>Primeiro item</li>
        <li>Segundo item item</li>

## Tabela:

`<table>`: Define a tabela

`<tr>`: Define as colunas

`<th>`: Define os conteúdos de cada coluna

Exemplo: 
    
    <table>

    <tr>
        <th>Itens</th>
        <th>Quantidade</th>
        <th>Total</th>
    </tr>
       
    <tr>
        <th>Registro 01</th>
        <th>3</th>
        <th>R$30</th>
    </tr>

    <tr>
        <th>Registro 02</th>
        <th>2</th>
        <th>R$30</th>
    </tr>

    </table>
